During my internship at **First Cloud AI Journey** from **April 17, 2026** to **July 30, 2026**, I had the opportunity to learn, practice, and apply cloud knowledge in a more realistic project environment. The internship helped me understand how a system is built from topic selection, requirement analysis, architecture design, infrastructure deployment, testing, and documentation.

## Self-Assessment Summary

| No. | Criteria | Level | Comment |

## Strengths

My main strength during the internship was persistence when debugging technical issues. With AWS, many problems are caused not only by code but also by service configuration, such as IAM permissions, CORS, environment variables, CloudFront cache, or region settings. I learned to check each layer, read CloudWatch logs, and document problems in a more systematic way.

I also improved my documentation skill. At first, my Worklog and Workshop content was too long, not well separated, or not written in a proper reporting style. After several revisions, I understood better how to describe my individual work, separate my contribution from team work, and format commands/configurations clearly.

## Areas for Improvement

I still need to improve planning earlier. In some stages, implementation, testing, and documentation were pushed close together, which made the final revision heavier. If I repeat a similar project, I will record daily notes more carefully so the final report can be prepared more smoothly.

I also need to continue improving team communication. In a project with many integration points like IRMS, backend, frontend, and infrastructure must agree on endpoint names, resource names, and data formats. Updating changes early and documenting technical decisions clearly is very important.

## Conclusion

Overall, I believe I completed my internship responsibilities well within my assigned scope. I did not build the entire project alone, but I made clear contributions to AWS infrastructure, AWS SAM, Cognito, API Gateway, Lambda/DynamoDB integration, EventBridge, SNS, CloudFront, frontend integration, deployment, testing, and documentation. This internship made me more confident with AWS and helped me better understand how a team completes a technical project.
