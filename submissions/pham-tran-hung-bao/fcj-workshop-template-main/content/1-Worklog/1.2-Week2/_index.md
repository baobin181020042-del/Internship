## Day 1 - 27/4: Learning Amazon VPC

**Work completed:**  On the first day of the second week, I studied Amazon Virtual Private Cloud (VPC) and learned the concept of a virtual private network in AWS. I became familiar with core networking components, including VPCs, Public Subnets, Private Subnets, Route Tables, Internet Gateways, and Elastic IPs. I also explored how these components work together to build a secure cloud network. In addition to the theoretical concepts, I practiced creating a new VPC, configuring a CIDR block, and designing subnets for different network segments.

**Result and lesson:** After this session, I understood the role of Amazon VPC in building cloud network infrastructure. I learned the purpose of each networking component and recognized how proper network design directly affects system scalability, availability, and security.
